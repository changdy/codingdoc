## 版本介绍
* Alpha
内部测试版本，一般不对外
* Beta
比Alpha稍好,对外测试版本
* RC *(Release Candidate)*
候选版本，测试后正确的话应该就是正式版本
* GA *(general availability)*
发型的稳定版本
* Release 
该版本意味“最终版本”，有时候只有一个R字符
* snapshot
自动更新版本
* spring 版本截图demo

## maven命令
* compile
编译
* package
打包，默认打包成jar
* clean
清除编译后产生的target之类
* install
可以理解为安装吧，最后会把当前项目装进本地仓库。（这样其他项目就可以使用）
